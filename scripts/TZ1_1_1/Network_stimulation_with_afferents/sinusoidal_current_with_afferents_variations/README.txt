# Система моделирования нейромышечной системы с синусоидальным воздействием и варьированием реципрокных связей

В данной директории расположены файлы для проведения симуляций нейромышечной системы с систематическим варьированием весов реципрокных синаптических связей при синусоидальном токовом стимулировании центрального паттерн-генератора.

## Подготовка симуляции: gen_cfg_sinusoidal_current_with_afferents_activity.py

В файле gen_cfg_sinusоидальным_current_with_afferents_activity.py формируется полная таблица параметров (cfg.csv), по которой в дальнейшем проводятся вычислительные эксперименты. Каждая строка этой таблицы задаёт одну уникальную конфигурацию модели, включающую параметры синусоидального тока для четырёх каналов ввода, веса одной целевой пары реципрокных связей и фиксированные веса всех остальных внутренних синаптических связей.

### Цель исследования

Изучение влияния асимметрии в реципрокных синаптических связях на способность центрального паттерн-генератора синхронизироваться с внешним синусоидальным ритмическим воздействием, а также анализ устойчивости и характеристик ритмической активности при варьировании силы прямого и обратного влияния в ключевых контурах торможения.

### Рассматриваемые контуры реципрокного торможения:

В эксперименте последовательно варьируются веса для четырёх пар реципрокных связей:
- CPG_N_Flex ↔ CPG_N_Ext — взаимодействие основных ритмогенерирующих нейронов ЦПГ, определяющее базовый ритм чередования фаз
- Ib_IN_Flex ↔ Ib_IN_Ext — интернейроны, получающие сигнал от сухожильных органов Гольджи, модулирующие активность ЦПГ в зависимости от мышечного усилия
- Ia_IN_Flex ↔ Ia_IN_Ext — интернейроны, получающие сигнал от первичных афферентов мышечных веретён, обеспечивающие чувствительность к длине мышцы
- R_Flex ↔ R_Ext — рефлексные нейроны переключения, участвующие в регуляции активности мотонейронов и стабилизации ритма

Для каждой целевой пары рассматриваются все комбинации весов прямого и обратного направления из набора [0.001, 0.5, 1.0], что позволяет исследовать режимы от практически одностороннего влияния до симметричного взаимодействия.

### Варьируемые параметры синаптических весов

Для каждой из четырёх целевых пар независимо варьируются:
- Вес прямой связи (например, CPG_N_Flex -> CPG_N_Ext): значения [0.001, 0.5, 1.0]
- Вес обратной связи (например, CPG_N_Ext -> CPG_N_Flex): значения [0.001, 0.5, 1.0]

Это даёт 9 комбинаций на одну пару. Все остальные 24 связи сети зафиксированы на уровне 0.5. Полный список фиксированных связей: Ia_IN_Ext->CPG_IN_Flex, CPG_IN_Flex->CPG_N_Flex, CPG_N_Ext->CPG_N_Flex, Ib_IN_Ext->Ib_IN_Flex, CPG_N_Flex->Ia_IN_Flex, R_Flex->Ia_IN_Flex, Ia_IN_Ext->Ia_IN_Flex, CPG_N_Flex->MN_Flex, Ib_IN_Flex->MN_Flex, R_Flex->MN_Flex, Ia_IN_Ext->MN_Flex, MN_Flex->R_Flex, R_Ext->R_Flex, Ia_IN_Flex->CPG_IN_Ext, CPG_IN_Ext->CPG_N_Ext, Ib_IN_Flex->Ib_IN_Ext, Ia_IN_Flex->Ia_IN_Ext, CPG_N_Ext->Ia_IN_Ext, R_Ext->Ia_IN_Ext, Ia_IN_Flex->MN_Ext, CPG_N_Ext->MN_Ext, Ib_IN_Ext->MN_Ext, R_Ext->MN_Ext, MN_Ext->R_Ext.

### Параметры синусоидального входного тока

Входной ток моделируется как синусоида с добавлением шума:
I(t) = base_current + amplitude * sin(2π * (t / period - phase)) + noise

Параметры варьируются для двух независимых групп каналов:
- Группа 0 (каналы 0 и 2, флексорная сторона): период, фаза, амплитуда
- Группа 1 (каналы 1 и 3, экстензорная сторона): период, фаза, амплитуда

В текущей конфигурации параметры зафиксированы: период [1 мс], фаза [0], амплитуда [5 для группы 0, 4 для группы 1]. Базовый ток для всех каналов установлен в ноль. При расширении эксперимента диапазоны могут быть изменены: период от 1 до 1000 мс, фаза от 0 до 1 (нормированная), амплитуда от 1 до 40 условных единиц.

### Параметры симуляции:
- Общая длительность симуляции: 10 секунд (10000 мс)
- Частота дискретизации: 5 точек в миллисекунду (шаг интегрирования dt = 0.2 мс)
- Входной ток: синусоидальный с заданными параметрами для каждого канала
- Уровень шума: NOISE_PERCENT = 0.05 (гауссовский шум со стандартным отклонением 5% от базового тока)
- Реципрокная схема: каналы 0 и 2 получают идентичные параметры, каналы 1 и 3 — идентичные параметры
- Афферентная обратная связь: фиксированные веса для каналов Ia (10) в инициализации модели

## Вычисление метрик: feature_extractor.py

В файле feature_extractor.py реализованы алгоритмы извлечения признаков из временных рядов. Ниже приведено описание всех вычисляемых метрик и их связь с переменными модели.

Метрики нейронов вычисляются по потенциалу V для каждого из 12 нейронов сети. neuron_{i}_V_mean — средний мембранный потенциал нейрона, показывает уровень деполяризации. neuron_{i}_V_std — стандартное отклонение потенциала, индикатор активности и колебаний. neuron_{i}_freq — средняя частота спайков в герцах, вычисляется как обратная величина к среднему межспайковому интервалу при превышении порога -20 мВ. neuron_{i}_cv — коэффициент вариации межспайковых интервалов, характеризует регулярность спайкинга: значения около нуля означают строгий ритм, выше 0.5 — хаотическую активность. Глобальные метрики V_mean_overall и V_std_overall дают обобщённую характеристику активности всей нейронной популяции.

Метрики переменной восстановления U также рассчитываются для каждого нейрона: neuron_{i}_U_mean и neuron_{i}_U_std описывают уровень и вариабельность адаптационных токов, которые в модели Ижикевича отвечают за пост-спайковую гиперполяризацию и утомление. Глобальные метрики U_mean_overall и U_std_overall характеризуют совокупный уровень адаптации в сети.

Метрики мышц вычисляются для сигналов силы F_flex и F_ext. Базовые статистики {muscle}_mean, max, min, std описывают средний уровень усилия, его амплитуду и стабильность. {muscle}_freq — частота мышечных сокращений, определяемая по интервалам между пиками силы. {muscle}_cv — регулярность этих сокращений. {muscle}_dom_freq — доминирующая частота в спектре мощности сигнала, вычисляемая методом Уэлча, позволяет выявить скрытые ритмы. Метрики flex_ext_ratio_mean и flex_ext_ratio_std характеризуют баланс между силами сгибателей и разгибателей. Корреляция F_flex_F_ext_corr показывает степень реципрокности: отрицательные значения указывают на чередование активации, положительные — на ко-активацию.

Метрики афферентов рассчитываются для шести сенсорных каналов модели.Limb.output. Для каждого канала aff_{i}_mean, std, max, min описывают интенсивность и вариабельность проприоцептивной обратной связи, поступающей от мышечных веретён и сухожильных органов.

Кинематические метрики вычисляются по углу сустава model.q и угловой скорости model.w. Для каждой величины рассчитываются среднее, стандартное отклонение, максимум и минимум, что позволяет оценить среднюю позу, амплитуду колебаний и стабильность движения. Корреляция q_w_corr характеризует фазовые соотношения: отрицательная корреляция типична для осцилляторных режимов.

Временной массив T используется для перевода индексов в реальное время при расчёте частот, а также для обрезки первых 50% данных с целью исключения переходных процессов.

На основе вычисленных метрик выполняется классификация режима работы системы на следующие типы: стабильный синхронный, ритмический, кластерный, хаотический, стабильный нейронный, ритмический нейронный и хаотический нейронный.

## Запуск симуляций: calc_sinusoidal_current_with_afferents_activity.py

Основной файл программы, выполняющий следующие этапы:

1. Загрузка конфигурации из cfg.csv
2. Инициализация полной модели: сеть Рыбака 2002 + афферентированная конечность + модели мышц с адаптацией, с предустановкой афферентных связей для каналов Ia
3. Для каждой конфигурации:
   - Парсинг целевой пары нейронов из поля target_pair с использованием класса SourceTargetParser
   - Установка весов для прямой и обратной связи в целевой паре согласно параметрам конфигурации
   - Генерация синусоидального входного тока с заданными параметрами периода, амплитуды и фазы для каждого канала
   - Запуск симуляции на 10 секунд с пошаговой записью всех переменных состояния
   - Обрезка первых 50% данных для исключения переходных процессов
   - Извлечение признаков из установившегося режима
   - Классификация режима работы
   - Сохранение временных рядов мышц (F_flex, F_ext) в отдельный CSV-файл с уникальным идентификатором
   - Добавление вычисленных метрик и статуса выполнения в итоговую таблицу

4. Параллельное выполнение всех симуляций с использованием joblib
5. Сохранение агрегированных результатов в единый файл

### Файл experiment_results_sinusoidal_currents_with_afferents.csv содержит:

- Все исходные параметры из cfg.csv (combination_id, target_pair, sine_period/amplitude/phase для каналов 0-3, base_current, noise_percent, веса варьируемой пары связей, веса 24 фиксированных связей)
- Статус выполнения симуляции (success/error)

- Классификация режима работы (mode)
- Все вычисленные метрики:
  * neuron_X_V_mean/std/freq/cv для каждого из 12 нейронов
  * neuron_X_U_mean/std для адаптационных переменных
  * F_flex/F_ext_mean/max/min/std/freq/cv/dom_freq
  * flex_ext_ratio_mean/std
  * aff_X_mean/std/max/min для шести афферентных каналов
  * q/w_mean/std/max/min
  * V/U_mean_overall/std_overall
  * F_flex_F_ext_corr, q_w_corr

### Файлы в data/:

Каждый файл имеет формат CSV с колонками:
- t — время в миллисекундах

- F_ext — сила экстензорной мышцы в условных единицах
